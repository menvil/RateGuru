# RateGuru Design Contract

## Reference

- Original source: `docs/design/reference/original/PlateRate.html`

## Direction

- Dark-first interface direction.
- Purple accent is the primary action and highlight color.
- Rounded cards define the main content surfaces.
- Feed views should use a card-based content model.
- Upload flows should follow a modal upload pattern.
- Detail views should follow a right drawer detail pattern.
